# 🍸 Cocktail Explorer

Una aplicación web moderna para explorar cócteles y sus recetas utilizando la API de TheCocktailDB.

## 🛠️ Tecnologías

- **Next.js 15** - Framework de React
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos utilitarios
- **TheCocktailDB API** - Datos de cócteles

## 📦 Instalación

```bash
# Clonar el repositorio
git clone <tu-repositorio>
cd mi-app

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
